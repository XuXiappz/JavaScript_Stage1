事件的三要素：
1.事件源：引发后续事件的事物（html标签)
2.事件：由js定义好的一些触发条件（例如点击，触摸，滚动等等）
3.事件驱动程序：（对样式html的操作，也就是dom操作）
事件扩展
1.获取事件源的方法：
1）document.getElementById("id名")
2) document.getElementsByClassName("类名")
3) documnet.getElementsByTagName("标签名")
2.事件的绑定：
1）匿名绑定：
div.onclick = function(){xxxx};
2) 函数名绑定：
function fn（）{xxxxx};
div.onclick = fn;
3) 行内绑定
<div id ="box" onclick="fn()"></div>
